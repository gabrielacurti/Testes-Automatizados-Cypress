# Testes Automatizados - Cypress

Projeto de automação de testes end-to-end utilizando Cypress, aplicado ao sistema de RH OrangeHRM (ambiente de demonstração).

## Objetivo
Validar cenários de login (positivos e negativos) através de testes automatizados, aplicando boas práticas como Page Object Model.

## Tecnologias
- Cypress
- JavaScript
- Git/GitHub

## Como rodar
```bash
git clone https://github.com/gabrielacurti/Testes-Automatizados-Cypress.git
cd Testes-Automatizados-Cypress
npm install
npx cypress open
```

## Cenários cobertos
- Login com credenciais válidas
- Login com credenciais inválidas
- Login com campos vazios
